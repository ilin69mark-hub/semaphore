# Семафор подсчёта на Go

Этот проект демонстрирует реализацию семафора подсчёта на языке Go. Семафор подсчёта позволяет ограничить количество потоков, которые могут получить доступ к ресурсу одновременно.

## Как запустить

1. Убедитесь, что у вас установлен Go (версия 1.16 или выше)
2. Выполните следующую команду в терминале:

```bash
cd /workspace
go run semaphore.go
```

## Особенности реализации

- Использует буферизованный канал для управления разрешениями
- Поддерживает таймауты для избежания блокировки
- Включает методы для проверки доступных разрешений
- Предоставляет возможность немедленного захвата/освобождения без ожидания
